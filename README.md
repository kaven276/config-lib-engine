概要说明
----------

* 可以集中收集各个应用的配置信息，如产品配置等等
* 方便配置进行版本控制(和库表保存比)
* 方便在不同环境部署不同的分支和版本(和库表保存比)
* 方便继续演进服务，如过滤，单条配置查询等等

支持配置文件的格式
----------------
* yaml/yml
* md/markdown
* json,json5
* csv,tsv

markdown 配置
--------------
* 可以随意些其他文档，直接忽略
* 支持 table 写法
* 取第一个 table 作为数据源
* table 表头作为每条记录的字段 key
* table 表头支持 name:num 方式，将字符串转换成数字返回
